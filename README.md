fontisizer.vim
==============

Font re-sizer vim code

Description
------------

Taken the font_sizer code from QMI TAKU and added support for linux type fonts.

Description
------------

To install

    git clone https://github.com/stephenmm/fontisizer.vim.git
    mkdir -p ~/.vim/plugin
    cp fontisizer.vim/fontisizer.vim ~/.vim/plugin
